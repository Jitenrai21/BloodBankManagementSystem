BLOOD BANK MANAGEMENT SYSTEM
Wireframes & UI Specifications
===============================
Phase 1 | Date: 2026-02-20

---

1. REGISTRATION PAGE (/users/register/)
----------------------------------------
Layout: Centered card form (max-width: 480px)

Fields:
  - Email Address       [text input]
  - Phone Number        [text input]
  - Role                [dropdown: Donor | Hospital]
  - Password            [password input]
  - Confirm Password    [password input]
  - [Register] button   [primary CTA]

Below form:
  - "Already have an account? Login" link

Validation:
  - Email format, phone format
  - Password strength indicator
  - Role selection required

---

2. LOGIN PAGE (/users/login/)
------------------------------
Layout: Centered card form (max-width: 400px)

Fields:
  - Email Address       [text input]
  - Password            [password input]
  - [Login] button      [primary CTA]

Below form:
  - "Forgot Password?" link
  - "Don't have an account? Register" link

---

3. ADMIN DASHBOARD (/users/dashboard/ → role=admin)
-----------------------------------------------------
Layout: Sidebar navigation + main content area

Sidebar Links:
  - Dashboard (overview)
  - Manage Users (approve/block)
  - Blood Inventory
  - Blood Requests
  - Donation Slots
  - Fraud Alerts
  - Reports

Main Content (overview cards):
  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
  │ Total Users │ │  Inventory  │ │  Requests   │ │ Fraud Flags │
  │     ###     │ │  ### units  │ │  ## pending │ │  ## active  │
  └─────────────┘ └─────────────┘ └─────────────┘ └─────────────┘

Recent Activity Table:
  | Timestamp | User | Action | Details |

---

4. DONOR DASHBOARD (/users/dashboard/ → role=donor)
-----------------------------------------------------
Layout: Sidebar + main content

Sidebar Links:
  - My Profile
  - Donation History
  - Book Donation Slot

Main Content:
  Profile Summary Card:
    - Name, Blood Group, Last Donation Date, Eligibility Status

  Donation History Table:
    | Date | Slot | Status | Units | Notes |

  Upcoming Slot (if booked):
    - Date, Time, Location, Status badge

---

5. HOSPITAL DASHBOARD (/users/dashboard/ → role=hospital)
----------------------------------------------------------
Layout: Sidebar + main content

Sidebar Links:
  - My Profile
  - Request Blood
  - My Requests

Main Content:
  Quick Stats:
    ┌───────────────┐ ┌───────────────┐
    │ Active Requests│ │ Fulfilled Req.│
    │      ##       │ │      ##       │
    └───────────────┘ └───────────────┘

  My Requests Table:
    | Date | Blood Group | Units | Urgency | Status |
  
  [+ New Blood Request] button

---

6. BLOOD REQUEST FORM (/requests/create/)
-------------------------------------------
Layout: Single-page form

Fields:
  - Blood Group         [dropdown: A+/A-/B+/B-/AB+/AB-/O+/O-]
  - Units Required      [number input, min:1]
  - Urgency             [radio: Normal | Urgent | Critical]
  - Patient Name        [text input, optional]
  - Required By Date    [date picker]
  - Reason / Notes      [textarea]
  - [Submit Request] button

---

7. INVENTORY PAGE (/inventory/)
---------------------------------
Layout: Table with filters

Filters (top bar):
  - Blood Group [dropdown]
  - Blood Type  [dropdown]
  - Available   [checkbox]
  - [Filter] button

Table Columns:
  | Blood Group | Type | Qty | Collection Date | Expiry Date | Available | Actions |

Actions (admin only):
  - Edit, Mark Unavailable

Expiry Alert Banner:
  ⚠ "X units expiring within 7 days"

---

8. FRAUD ALERTS PAGE (/fraud/) [Admin Only]
---------------------------------------------
Layout: Table with severity badges

Table Columns:
  | Time | User | Flag Type | Severity | Risk Score | IP | Resolved | Action |

Severity Badges:
  - 🔴 High  🟡 Medium  🟢 Low

Actions:
  - [Resolve] button per row
